## 🚀 Quick Start para Contribuir

```bash
# 1. Fork o projeto
# 2. Clone seu fork
git clone <seu-fork>

# 3. Crie uma branch
git checkout -b feature/minha-feature

# 4. Desenvolva e teste
npm run dev
npm test

# 5. Commit seguindo padrões
git commit -m "feat: adiciona nova feature"

# 6. Abra Pull Request
```
